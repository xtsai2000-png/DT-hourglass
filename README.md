# 沉光（DT-hourglass）

> 數位化時光沙漏 — 在時間的流動中，凝聚生命的美麗與智慧

---

## 核心理念

你是否曾感受到，生命中的每一秒，都在我們周遭的細微震動、身心的律動中，悄然感知著環境的低語與身心的起伏？

「沉光」邀請你，將這些無形的生命共鳴，凝聚為有形的「時光沉積」。

---

## 它是什麼

每一秒鐘，隨著你佩戴的智慧裝置，感知環境的細微變化與身心的律動，經由 AI 的巧妙編織，這些訊號將轉化為擁有一顆顆擁有獨特光澤、形態與質地的「時光微塵」——

- 🔵 **剔透藍** — 代表寧靜
- ⚫ **渾濁灰** — 代表焦慮
- 🟣 **深邃紫** — 代表高度專注

微塵在螢幕上緩緩落下，它們不只是堆疊，而是在進行一場持續一生的「時光彩繪」。

每當夜幕低垂，你的沙漏底部，將不再是普通的沙堆，而是深邃、層次豐富的「時光沉積」。

---

## 核心特色

每一層顏色與質地，都如同寶石般閃耀，象徵著不同生命時段的重量。

**你可以隨時回溯某一層顏色**，AI 將帶領你重新體會那一段時光的氛圍：

> 那層深紫色，或許代表著你那晚全神貫注地解決複雜問題，思緒高度集中的時刻。

---

## 功能藍圖

- 🕐 **數位沙漏視覺化** — 時間流逝的動態景觀
- 🤖 **AI 時光微塵生成** — 根據生理/環境訊號賦予顏色與質地
- 📊 **時光沉積回溯** — 隨時回顧過去某段時光的氛圍
- 🎨 **生命地層學** — 你是獨一無二的時間沉積藝術家
